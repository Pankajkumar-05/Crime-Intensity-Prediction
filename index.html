Case notes — how this was built

The source data is a set of 9,919 first-information-style crime reports from 29 Indian cities (2020), covering victim demographics, weapon used, time of occurrence, and case outcome. Each record was labelled high intensity if its crime type was one of eight serious categories — homicide, assault, sexual assault, robbery, kidnapping, domestic violence, arson, or firearm offense — and low intensity otherwise.

Three classifiers were benchmarked — logistic regression, random forest, and gradient boosting — on features including hour, month, day of week, city, victim age, victim gender, and weapon used. Because high-intensity cases were a minority class, the training set was rebalanced by upsampling before the final model was fit.

The query form above runs the same filtering logic as the notebook's interactive predictor: it looks up matching historical cases for your selected city, crime type, and victim gender, and reports the model's predicted intensity split across those cases.
